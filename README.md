# 🍽️ ChefSync Enterprise

ChefSync Enterprise adalah aplikasi Restaurant Management System berbasis Flutter dan Firebase yang dirancang untuk membantu operasional restoran secara real-time.

## ✨ Features

- Authentication
- Dashboard
- Restaurant Ordering
- Kitchen Display System (KDS)
- Waiter Module
- Cashier Module
- Receipt
- Firebase Firestore Realtime

---

## 🛠 Tech Stack

- Flutter
- Riverpod
- GoRouter
- Firebase Authentication
- Cloud Firestore

---

## 🏗 Architecture

Feature First Architecture

Repository Pattern

Riverpod State Management

Realtime Firestore
