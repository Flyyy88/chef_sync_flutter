## Key Features Included:
- **Role-Based Access**: Multi-role support (Admin, Manager, Cashier, Waiter, Kitchen).
- **Riverpod Providers**: Scalable state logic, async data updates, state monitoring.
- **Material 3 UI Theme**: Light/Dark adaptive styling using elegant color scales.
- **Floor Seat Layout**: Visual interactive table models.
- **KDS (Kitchen Display System)**: Preparation status trackers, real-time ticket timelines.
- **Supplier & Stock Tracking**: Automated low-stock thresholds.

## Setup Instructions:
1. Ensure Flutter is installed: `flutter --version`
2. Navigate to root folder and fetch packages: `flutter pub get`
3. Run code generators for Riverpod annotations (if compiling generated state): `dart run build_runner build --delete-conflicting-outputs`
4. Run app on your connected device/simulator: `flutter run`
